# Double Pendulum Simulation
A real-time simulation of a **double pendulum**, a classic chaotic system where small changes in initial conditions lead to unpredictable motion.

https://github.com/user-attachments/assets/61f63be1-9082-4bdb-91e8-d645dc228807

## Features
- Realistic physics using coupled equations of motion  
- Smooth rendering with Raylib  
- Motion trail visualization  
- Video recording using FFmpeg  

## Controls
- `SPACE` → Reset simulation  
- `R` → Start recording (20 sec)  
- `ESC` → Exit  

## Requirements
- Raylib  
- FFmpeg

```bash
bash build.bash
./a.out
